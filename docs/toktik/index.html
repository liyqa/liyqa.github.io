    小音 APP

    2022/10/29

    # 缘起

           抖音APP在商业上是如此的成功，以至于本人也想尝试来做一个山寨版的抖音APP，于是，小音APP 就诞生了。

           相比抖音154MB的体积，小音APP 非常轻量，4MB的体积，全部采用原生开发，保证了APP的高性能。 虽然功能上还无法媲美抖音，却完成了刷视频这个最核心的需求，并且默认采用自动刷的模式 （小音APP 保留了切换自动刷和手动刷的选项）。

    # 应用商店

           目前，国内各大应用商店的入口，都没有对独立开发者开放，要么就是需要软著，反正就是有各种障碍。 本人嫌麻烦，也没钱来上架APP，因此，小音APP 只能在本站下载。

           至于有些用户担心APP有病毒之类的，这一点大可放心。本人的水平还不足以写出有病毒的APP（听说， 会写病毒的都是高手，敝人水平有限，只会写点CRUD之类的东西），况且，手机上也有各种杀毒软件， 实在是不放心的小伙伴们，可以杀杀毒，确认安全后再来使用。

    # 商业化

           抖音APP，本人可以复制，但是抖音的商业模式，本人复制不了，这也不是技术就能解决的问题了。

           关于软件开发这个行业，有优势也有劣势。        优势在于开发软件基本上已经摆脱了对地域的限制，也就是说，无论你在哪里， 只要有台能联网的电脑，就可以开始工作了（当然了，投资理财之类的炒股炒基金也是一样， 只需要一部能上网的手机就行了），这对于希望自由支配时间的小伙伴来说，简直就是最理想的职业了；
           劣势在于现在的软件基本上都是免费的，比如微信不要钱就能用，抖音也是不要钱就能用。 换句话说就是，即便是你自己做出了一个微信，抖音也没啥用，还是赚不到钱， 最后的结果就是技术沦为了给资本打工的工具，况且资本还有35岁清退你这个终极杀手锏， 这就进一步挤压了行业内从业人员的生存空间。敝人只是个微不足道的小人物，改变不了这个现状。
           网络上流传的那句话，宇宙的尽头是公务员，至少在国内是千真万确的。
